# Mesh Chat

A privacy-centric, peer-to-peer messaging application utilizing Bluetooth Low Energy mesh networking and end-to-end encryption.

## Features

- **Bluetooth Mesh Networking**: Communicate without internet connection
- **End-to-End Encryption**: Military-grade message protection
- **Decentralized**: No central servers, no data harvesting
- **Offline Capable**: Works entirely offline via BLE mesh
- **Privacy First**: No phone numbers or personal data required

## Tech Stack

- Swift
- SwiftUI
- CoreBluetooth
- CryptoKit
